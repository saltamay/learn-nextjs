# Welcome to React Tutorial!

This is an interactive tutorial that is composed with guides and challenges to help you learn React and its concepts step by step.

The goal of this tutorial is to quickly give you an experience of what it feels like to work with React, right in the browser.
It does not aim to be comprehensive, and you don't need to understand everything before moving on. All playgrounds are editable with a full-featured editor and dev environment, so you can play with the code and see the result right away and learn by doing.

## What's React?

React is a JavaScript library for building user interfaces. 

## Getting Started

With this tutorial, we assume you are already familiar with the basic concepts of HTML, CSS and JavaScript.
